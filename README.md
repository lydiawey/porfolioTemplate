# Mi porfolio template, UDIT 25-26
> by lydiawey

You can find the deployed site here : [https://lydiawey.github.io/porfolioTemplate/] (https://lydiawey.github.io/porfolioTemplate/)

Portfolio personal desarrollado como proyecto académico.

## Descripción
Web portfolio con enfoque visual y scroll-based, donde se muestran proyectos de diseño gráfico y editorial.

## Tecnologías
- HTML5
- CSS3
- Bootstrap 5
- GSAP
- JavaScript

## Instalación
No requiere instalación. Abrir `index.html` o usar GitHub Pages.

## Uso de IA
Durante el desarrollo se utilizó asistencia de IA como apoyo puntual para:
- resolución de dudas técnicas
- refactorización de CSS
- estructuración de componentes

Las decisiones finales de diseño y contenido fueron propias.
